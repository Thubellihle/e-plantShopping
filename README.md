# 🌿 Paradise Nursery

Paradise Nursery is an online plant shopping application where users can browse houseplants, add them to a cart, and manage their selections.

## 🚀 Features
- Browse plants by category
- View plant details (image, name, price)
- Add items to cart
- Increase/decrease quantity
- Remove items from cart
- View total price
- Responsive navigation bar

## 🛠️ Technologies Used
- React
- Redux Toolkit
- CSS

## 📦 Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
